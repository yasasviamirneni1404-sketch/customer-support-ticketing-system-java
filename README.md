# Customer Support Ticketing System (Core Java)

## Overview
A Java-based console application designed to simulate real-world customer support ticket workflows and replace manual issue tracking.

## Features
- Create support tickets
- View all tickets
- Update ticket status
- Close tickets

## Tech Stack
- Java (Core Java)
- OOP Principles
- Collections Framework

## How to Run
1. Clone the repository
2. Compile and run TicketApplication.java
3. Follow the menu options in the console

## Learning Outcome
- Hands-on experience with Core Java and OOP
- Understanding of backend business logic
- Error handling and structured application design
